# Una Nueva Constitución Chilena a la luz del Poder Constituyente

**Autor(es):** Oro Mancilla, Felipe

**Año:** 2022  **Tipo:** TDPRE

## Resumen

El presente trabajo busca exponer el origen, fundamento y titularidad de los principios de soberanía popular y poder constituyente, para mostrar como el pueblo es quien tiene la labor de crear una constitución. A sí mismo el trabajo, luego contrasta dichos valores con el proceso constituyente que se lleva a cabo en Chile, estudiando detalladamente el itinerario propuesto por el gobierno. Por último se señala cuál es el mecanismo que mejor reflejaría los postulados de una teoría constituyente para obtener una carta fundamental legítima.
