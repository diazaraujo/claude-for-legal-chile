# Guía Formulación Ética de Proyectos de Ciencia de Datos

**Autor(es):** Universidad Adolfo Ibáñez, División de Gobierno Digital

**Año:** 2022  **Tipo:** Otro

## Resumen

El objetivo de esta guía es proveer una base de conocimiento mínima para la formulación ética de proyectos en ciencia de datos conociendo los riesgos a la hora de formular y entrenar modelos y/o en el proceso de compra del mismo por el sector público.
