# El cohecho como propuesta o aceptación de una donación remuneratoria

**Autor(es):** Mañalich Raffo, Juan

**Año:** 2019  **Tipo:** Artículo de revista

## Resumen

La dogmática del cohecho suele
recurrir a la noción de un pacto ilícito,
a cuya celebración se orientaría el
comportamiento del funcionario, y que
exhibiría la forma de un (pseudo-)contrato
sinalagmático. Aquí se defenderá la superioridad del modelo alternativo
de la “donación remuneratoria”, sobre
cuya base se esclarecerá la relación que
debe darse entre el beneficio solicitado o
aceptado y la ejecución u omisión de una
acción conectada con el cargo, así como
el grado de determinación con el cual
debería quedar especificado tal “servicio
funcionarial”.
