# Ley 21.394 — Modernización Notarial

> **Borrador no validado.** Pendiente de revisión por abogado civil /
> notarial.

## Resumen

La Ley 21.394 (2021) **moderniza el sistema notarial chileno** + el de
Conservadores de Bienes Raíces + Archiveros Judiciales. Modifica el
**Código Orgánico de Tribunales (COT)** introduciendo:

1. **Tramitación electrónica** ante notarías.
2. **Firma electrónica avanzada** para escrituras.
3. **Registro electrónico notarial** centralizado.
4. **Plazos reducidos** para inscripciones.
5. **Mayor competencia + transparencia** entre notarías.
6. **Régimen tarifario** ajustado.

## Componentes clave

### Tramitación electrónica

- **Solicitudes** vía plataforma electrónica.
- **Pago electrónico** de derechos.
- **Recepción de copias autorizadas** vía PDF firmado.

### Firma electrónica avanzada

- **Escrituras públicas** firmadas electrónicamente con FEA del
  cliente + del notario.
- **Cruce con Ley 19.799** + reglamentación post-21.394.

### Registro electrónico notarial

- **Centralizado** + interoperable.
- **Búsqueda pública** de instrumentos.
- **Consulta abierta** de inscripciones.

### Plazos reducidos

- **Inscripciones** en plazos menores que el régimen tradicional.
- **Procedimientos** estandarizados.
